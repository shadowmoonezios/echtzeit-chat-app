# Echtzeit Chat App

Willkommen bei der Echtzeit Chat App! Dies ist eine Full-Stack-Anwendung, die Benutzern ermöglicht, in verschiedenen Kanälen zu chatten und in Echtzeit zu kommunizieren.

## Funktionen
- Benutzerregistrierung und -anmeldung
- Unterstützung für mehrere Chat-Kanäle
- Echtzeit-Message-Update mit WebSockets
- Benutzerprofile mit Avatar und Status
- Responsive Design für mobile Nutzung

## Technologien
- **Frontend:** React.js
- **Backend:** Node.js mit Express
- **Datenbank:** MongoDB
- **Echtzeit-Kommunikation:** Socket.IO

## Installation
1. Klone das Repository:
   ```bash
   git clone https://github.com/benutzer/echtzeit-chat-app.git
   ```
2. Wechsle in das Verzeichnis:
   ```bash
   cd echtzeit-chat-app
   ```
3. Installiere die Abhängigkeiten:
   ```bash
   npm install
   ```
4. Starte die Anwendung:
   ```bash
   npm start
   ```
